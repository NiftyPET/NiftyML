NiftyML: PET-MR with Machine Learning
=====================================

Installation
------------

.. code:: sh

    pip install niftyml


Usage
-----

.. code:: python

    from niftypet import ml
    ml.dcl2021(...)

See `examples <./examples/>`_ for more.
